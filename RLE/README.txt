О JS файлах:
- core.js основной файл. Его надо запускать 
- escape.js файл для работы метода Escape
- jump.js файл для работы метода Jump

О TXT файлах:
- input.txt файл с исходным текстом, который надо закодировать
- output.txt файл с закодированным текстом
- result.txt файл с декодированным текстом

Формат:
- node core.js (encode/decode) (escape/jump) (input.txt output.txt/output.txt result.txt)
